# Smart Route Planning Game
An interactive route planning game developed as part of an Algorithms and Data Structures (ADSA) project. This application visualizes graph-based pathfinding concepts and allows users to explore optimal routes using real-time algorithm execution.

## Features
* Interactive graph visualization for route planning
* Shortest path computation using Dijkstra’s Algorithm
* AI-based suggestions for optimized routes
* Real-time graph updates and dynamic path calculation
* Audio feedback for user interactions
* Full-stack architecture with REST API support

## Tech Stack

### Backend
* Java 17
* Spring Boot
* Maven
* Graph Algorithms (Dijkstra’s Algorithm)

### Frontend
* React
* JavaScript
* CSS
* Create React App

## Installation & Setup

### Prerequisites

* Java 17 or higher
* Node.js (v14 or higher)
* npm or yarn

### Backend Setup

cd Backend/demo
.\mvnw.cmd spring-boot:run

Backend runs on:
http://localhost:8080

### Frontend Setup
cd Frontend
npm install
npm start

Frontend runs on:
http://localhost:3000

## Usage
1. Start both backend and frontend servers
2. Open the application in your browser
3. Interact with the graph interface
4. Select source and destination nodes
5. Visualize the shortest path
6. Use AI suggestions to optimize routes

## Purpose
This project demonstrates how graph algorithms like Dijkstra’s can be applied in real-world route optimization, making complex concepts easier to understand through visualization and interaction.

## Note
This project is developed for educational purposes to demonstrate algorithmic concepts in an interactive way.
